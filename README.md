# Pusat Borang PERKESO Keningau 📋

Laman web pusat muat turun borang-borang rasmi **PERKESO** untuk Pejabat Keningau, Sabah. Semua borang disusun mengikut kategori dengan pautan terus ke laman rasmi [perkeso.gov.my](https://www.perkeso.gov.my).

![Kategori](https://img.shields.io/badge/Kategori-15-blue) ![Borang](https://img.shields.io/badge/Borang-88-green) ![Pautan](https://img.shields.io/badge/Pautan-100%25%20Rasmi-yellow)

## ✨ Ciri-ciri

- 🔍 **Carian langsung** dengan penyorotan kata kunci
- 🏷️ **Penapis kategori** (chip) — dijana automatik dari data
- 🎨 **Lencana berwarna** mengikut jenis dokumen (PDF, Senarai Semak, Panduan, Rujukan)
- 📱 Sepenuhnya **responsif** (desktop, tablet, telefon)
- 📞 Maklumat pejabat Keningau (alamat, telefon, jam operasi)
- ⚡ Laman **statik** — laju & tiada pelayan diperlukan

## 📁 Struktur

```
.
├── index.html    # Struktur laman
├── style.css     # Reka bentuk (warna rasmi PERKESO)
├── script.js     # Data borang + logik carian/penapis

```

## 🚀 Cara Guna

1. **Lokal**: Klik dua kali `index.html` → terbuka dalam pelayar.
2. **Atas talian**: Aktifkan GitHub Pages (lihat di bawah) untuk akses URL awam.

## 🌐 Aktifkan GitHub Pages

1. Pergi ke **Settings → Pages** dalam repo ini
2. **Source**: `Deploy from a branch`
3. **Branch**: `main` / `(root)` → **Save**
4. Tunggu ~1 minit, laman akan tersedia di:
   ```
   https://kamkikorich.github.io/Borang-Perkeso/
   ```
